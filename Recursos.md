Recursos recomendados para aprender ciberseguridad

Libros:
- "The Web Application Hacker's Handbook" — Dafydd Stuttard & Marcus Pinto
- "Practical Malware Analysis" — Michael Sikorski & Andrew Honig

Cursos y plataformas:
- TryHackMe (https://tryhackme.com)
- Hack The Box (https://www.hackthebox.com)
- OWASP WebGoat para practicar vulnerabilidades web

Herramientas útiles:
- nmap: escaneo de puertos
- wireshark: análisis de tráfico de red
- burp suite: pruebas de seguridad web
- metasploit: explotación y pruebas de penetración

Comunidades y documentación:
- OWASP (https://owasp.org)
- CERT/INCIBE (en español)
